# ✈️ Анализ цен на авиабилеты

Данный проект заключается в комплексном анализе датасета: первичном осмотре, очистке данных, SQL-анализе, создании визуализаций, интерпретации результатов. В датасете содержатся данные о **бронировании авиабилетов**.

*Источник датасета:* [Flight Price Prediction – Kaggle](https://www.kaggle.com/datasets/shubhambathwal/flight-price-prediction)
